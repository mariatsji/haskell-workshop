# haskell-workshop

Welcome!

## Part 1 - Setup
- Install Stack as a Haskell build tool : https://docs.haskellstack.org/en/stable/README/#how-to-install
- Clone this repository

    `cd haskell-workshop`

    `stack --help`

    `stack setup`

    `stack build`

    `stack test`

    `stack exec haskell-workshop`

    `stack repl`

    `> 1 + 1`

    `> :quit`
